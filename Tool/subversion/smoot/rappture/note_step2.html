Initial Observations

In this step, enter the initial observations of your system in the following fields:

  • Observed Inputs and Observed Outputs: Upload your observations in the upper loader menu. Upload one CSV file for observed inputs and one CSV file for observed outputs. Search the wiki page for information about the correct CSV file formatting
  • Lower Input Bounds and Upper Input Bounds: Turn on the Input Bounds control and enter the lower and upper bounds of the inputs in a tuple form (e.g. "0,0" and 1,1" for 2 variables, both with a lower bound of 0 and an upper bound of 1). Also, turn off the control and upload a set of initial designs of your experiment in a CSV file in the upper loader menu, along with your observations. Search the wiki page for information about the correct CSV file formatting
  • Number of Iterations: Enter the maximum number of iterations for the program

After entering all inputs, proceed to the next step
